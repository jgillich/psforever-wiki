category, linked at bottom.

## 2003

### May

- [1.0.0](1.md.0.0) - [PlanetSide](PlanetSide.md)
  Launches on May 20th!
- [1.1](1.md.1) - [Jamming](Jammer_Grenade.md) of
  friendly [Combat Engineering](Combat_Engineering.md)
  disabled. [Implant](Implants.md) confirmation removed.
  [Tells](Tell.md) no longer case sensitive.
- [1.2](1.md.2) - [Zone](Zone.md) crash fixes, weather
  tweaks, and [Server](Server.md) listing tweaks added. Base
  kill [Experience](Experience.md) doubled from 50 to 100.
- [1.3](1.md.3) - Zone and game crash fixes

### June

- [1.4](1.md.4) - Vehicle Queues, Population Lock indicators,
  and Bandwidth settings. Other minor tweaks.
- [1.5](1.md.5) - [Skyguard](Skyguard.md),
  [Liberator](Liberator.md)
  _[Certification](Certification.md)_, and [Orbital
  Strikes](Orbital_Strike.md)/[EMPs](EMP.md) go live!
  Respawn into hacked bases available. [Hotspot](Hotspot.md)
  improvements.

### July

- [Liberator
  Patch](Unnumbered_Patches.md#Liberator_.287-3-03.29) -
  [Liberator](Liberator.md) goes live and is available for
  purchase! Door bug fixes.

### August

- [LLU
  Patch](Unnumbered_Patches.md#LLU_Goes_Live_-_August_14.2C_2003) -
  LLU capture system goes live! [Lasher](Lasher.md) and
  [Starfire](Starfire.md) damage degredation tweaks,
  [Basilisk](Basilisk.md) and [Wraith](Wraith.md)
  combined into single [ATV](ATV.md) cert, and HART waypoint
  exploit fixes added.
- [1.8.8](1.md.8.8) - LLU changes, ATV down to 2 cert points,
  Starfire MAX projectile matches Sparrow MAX speed
- [1.9.1](1.md.9.1) - Flashing LLU link, faster auto-drive off
  vpad, respawning no longer costs NTU
- [1.9.2](1.md.9.2) - Speedhacking detection hotfix

### September

- [1.10.10](1.md.10.10) - Bug fixes, lockers/implants/cert
  terminals hackable, equipment favorites names extended

### October

- [1.11.14](1.md.11.14) - Balance Pass and Platoons go live!
- [2.0.47](2.md.0.47) - Lattice link map hotfix

### November

- [2.0.49](2.md.0.49) - AMS zone crash fix, drop pod client
  crash fix, extra health exploit fix
- [2.1.5](2.md.1.5) - Client startup crash fix
- [2.1.6](2.md.1.6) - Ancient Tech weapons and vehicle
  tweaks/fixes. Galaxies banished from caverns, VS/TR MAX tweaks, and
  ACE tweaks.
- [2.2.13](2.md.2.13) - Keymapping tweaks, hack time and LLU
  lifespan available on map, geowarp tweaks, and module tweaks
  (including ability to spawn all types).
- [2.2.15](2.md.2.15) - Server crash fixes, animation culling
  and backpack cleanup code tweaks.

### December

- [2.3.6](2.md.3.6) - Battle Concentration Initiative (server
  merge) and Lodestar go live! HART timer changed to 5 minutes.
- [2.3.9](2.md.3.9) - Lodestar tweaks, 12 hour empire timer,
  memory and muzzle flash fixes.
- [2.3.13](2.md.3.13) - Spawning in the AMS bubble instead of
  tube, zone/client crash fixes, cavern/geowarp fixes
- [2.3.14](2.md.3.14) - Medic zone crash and striker client
  crash fixes
- [2.3.18](2.md.3.18) - Server crash fixes
